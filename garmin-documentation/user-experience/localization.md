# Localization - User Experience Guidelines

## Overview

This page covers internationalization for Connect IQ applications.

## Key Topics

Localization includes:
- String resources
- Language support
- Cultural considerations
- Date and time formats
- Number formatting
- Text expansion
- RTL support

## Best Practices

- Externalize strings
- Support common languages
- Test with translated text
- Consider text length
- Cultural sensitivity
- Units and measurements

**Note:** For complete localization guidelines with internationalization best practices, visit the Garmin Developers portal at developer.garmin.com/connect-iq/user-experience-guidelines/localization/
