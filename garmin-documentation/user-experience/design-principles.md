# Design Principles - User Experience Guidelines

## Overview

This page covers core design philosophy for Connect IQ applications.

## Key Design Principles

Design principles typically include:
- User-centered design
- Simplicity and clarity
- Consistency
- Accessibility
- Performance
- Context-awareness

## Application to Garmin Devices

- Small screen optimization
- Outdoor visibility
- Glanceable information
- Battery efficiency
- Athletic focus
