# Entry Points - User Experience Guidelines

## Overview

This page covers app entry and navigation patterns for Connect IQ applications.

## Key Topics

Entry points include:
- App launch mechanisms
- Widget entry
- Glance entry
- Complication taps
- Intent handling
- Deep linking

## Best Practices

- Clear entry affordances
- Consistent behavior
- Fast loading
- Appropriate context
- Graceful errors
