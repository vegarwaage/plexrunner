# Map Views - User Experience Guidelines

## Overview

This page covers map interface design for Connect IQ applications.

## Key Topics

Map view design includes:
- Map types and styles
- Overlay design
- Interaction patterns
- Zoom and pan
- Data visualization
- Performance considerations

## Best Practices

- Clear map data
- Appropriate detail levels
- Efficient rendering
- User location indication
- Context-appropriate styling

**Note:** For complete map view design guidelines, visit the Garmin Developers portal at developer.garmin.com/connect-iq/user-experience-guidelines/map-views/
