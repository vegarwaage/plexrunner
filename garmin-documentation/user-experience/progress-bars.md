# Progress Bars - User Experience Guidelines

## Overview

This page covers progress indication design for Connect IQ applications.

## Key Topics

Progress bar design includes:
- Progress types
- Visual styles
- Indeterminate vs determinate
- Loading states
- Feedback timing
- Cancellation options

## Best Practices

- Show for long operations
- Provide time estimates
- Allow cancellation when possible
- Clear completion states
- Appropriate styling

**Note:** For complete progress bar design guidelines, visit the Garmin Developers portal at developer.garmin.com/connect-iq/user-experience-guidelines/progress-bars/
