# Watch Faces - User Experience Guidelines

## Overview

This page covers watch face design guidelines for Connect IQ.

## Key Topics

Watch face design includes:
- Layout and composition
- Data hierarchy
- Readability
- Always-on considerations
- Battery optimization
- Customization options

## Design Considerations

- Time display prominence
- Data field selection
- Color and contrast
- Screen shape adaptation
- AMOLED optimization

**Note:** For complete watch face design guidelines with best practices and examples, visit the Garmin Developers portal at developer.garmin.com/connect-iq/user-experience-guidelines/watch-faces/
