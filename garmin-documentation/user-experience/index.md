# User Experience Guidelines - Connect IQ

## Overview

This section provides design best practices for Garmin devices and Connect IQ applications.

## Available Topics

The User Experience Guidelines section includes:

- **Design Principles** - Core design philosophy
- **Understanding What You Are Building** - Project scope and goals
- **Developing the Concepts** - Concept development
- **Designing Workflows and Interactions** - Interaction design
- **Incorporating the Visual Design and Product Personalities** - Visual design
- **Entry Points** - App entry and navigation
- **Views** - View design patterns
- **Watch Faces** - Watch face design guidelines
- **Data Fields** - Data field design
- **Menus** - Menu design patterns
- **Confirmations** - Confirmation dialogs
- **Progress Bars** - Progress indication
- **Map Views** - Map interface design
- **Localization** - Internationalization

**Note:** For complete User Experience Guidelines with design patterns and examples, visit the Garmin Developers portal at developer.garmin.com/connect-iq/user-experience-guidelines/
