# Confirmations - User Experience Guidelines

## Overview

This page covers confirmation dialog design for Connect IQ applications.

## Key Topics

Confirmation design includes:
- When to use confirmations
- Dialog structure
- Button placement
- Clear messaging
- Default actions
- Cancellation flows

## Best Practices

- Use sparingly
- Clear, concise text
- Appropriate for action severity
- Easy to dismiss
- Consistent patterns

**Note:** For complete confirmation dialog guidelines, visit the Garmin Developers portal at developer.garmin.com/connect-iq/user-experience-guidelines/confirmations/
