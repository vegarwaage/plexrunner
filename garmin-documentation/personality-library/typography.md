# Typography - Personality Library

## Overview

This page covers font and text styling for Connect IQ applications.

## Key Topics

Typography guidelines include:
- System fonts
- Font sizes
- Font weights
- Line height
- Text alignment
- Custom fonts
- Readability

## Best Practices

- Use system fonts for consistency
- Appropriate sizing for device
- Sufficient contrast
- Readable at a glance
- Support for multiple languages
