# Colors - Personality Library

## Overview

This page covers color palettes and usage guidelines for Connect IQ applications.

## Key Topics

Color guidelines include:
- Primary color palettes
- Secondary colors
- Accent colors
- Background colors
- Text colors
- Status colors
- Accessibility considerations

## Best Practices

- Use Garmin brand colors appropriately
- Ensure sufficient contrast
- Support AMOLED displays
- Consider outdoor visibility
- Accessibility compliance

**Note:** For complete color palette documentation with hex values and usage guidelines, visit the Garmin Developers portal at developer.garmin.com/connect-iq/personality-library/colors/
