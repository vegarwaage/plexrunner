# Prompts - Personality Library

## Overview

This page covers user prompt design for Connect IQ applications.

## Key Topics

Prompt design includes:
- Prompt types
- Message structure
- Button placement
- Tone and voice
- Timing
- Dismissal patterns

## Best Practices

- Clear, actionable text
- Appropriate timing
- Easy to understand
- Consistent styling
- Respectful of user attention

**Note:** For complete prompts documentation with design patterns and examples, visit the Garmin Developers portal at developer.garmin.com/connect-iq/personality-library/prompts/
