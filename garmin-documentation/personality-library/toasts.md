# Toasts - Personality Library

## Overview

This page covers toast notification design for Connect IQ applications.

## Key Topics

Toast design includes:
- Toast types
- Duration timing
- Position on screen
- Styling
- Dismissal behavior
- Content guidelines

## Best Practices

- Brief messages
- Appropriate duration
- Non-intrusive placement
- Clear visibility
- No critical information

**Note:** For complete toasts documentation with design specifications and usage patterns, visit the Garmin Developers portal at developer.garmin.com/connect-iq/personality-library/toasts/
