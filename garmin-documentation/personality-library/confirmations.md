# Confirmations - Personality Library

## Overview

This page covers confirmation patterns for Connect IQ applications.

## Key Topics

Confirmation patterns include:
- Confirmation dialog structure
- Button ordering
- Default selections
- Destructive actions
- Cancellation
- Visual styling

## Best Practices

- Use for important actions
- Clear consequences
- Standard button placement
- Easy to cancel
- Consistent patterns
