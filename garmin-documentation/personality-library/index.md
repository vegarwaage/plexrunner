# Personality Library - Connect IQ

## Overview

The Personality Library provides design resources and UI component specifications for Connect IQ applications.

## Available Topics

The Personality Library section includes:

- **Colors** - Color palettes and usage
- **Iconography** - Icon design and usage
- **Typography** - Font and text styling
- **Input Hints** - Input guidance patterns
- **Prompts** - User prompt design
- **Confirmations** - Confirmation patterns
- **Toasts** - Toast notification design
- **Action Views** - Action-oriented views
- **Page Loops** - Page navigation patterns
- **Progress Bars** - Progress indication components

**Note:** For complete Personality Library documentation with design resources and component specifications, visit the Garmin Developers portal at developer.garmin.com/connect-iq/personality-library/
