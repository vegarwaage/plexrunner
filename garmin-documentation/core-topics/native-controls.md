# Native Controls - Connect IQ

## Overview

This page covers built-in UI components available in Connect IQ development.

## Key Topics

Native Controls provide:
- Pre-built UI components
- Standard interface elements
- Platform-consistent design
- Reduced development time
- Device-optimized rendering

Common native controls:
- Pickers
- Text inputs
- Progress bars
- Menus
- Buttons
