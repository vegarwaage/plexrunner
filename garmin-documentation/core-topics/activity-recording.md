# Activity Recording - Connect IQ

## Overview

This page covers fitness tracking features in Connect IQ applications.

## Key Topics

Activity recording includes:
- Session management
- FIT file generation
- Data field recording
- Lap marking
- Auto-pause
- Activity summary

## Recordable Data

- Time and distance
- Heart rate
- Speed and pace
- Elevation
- Custom data fields
- Sensor data

## Related Topics

- Sensors
- Positioning
- Quantifying the User
- Persisting Data
