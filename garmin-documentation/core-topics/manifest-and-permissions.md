# Manifest and Permissions - Connect IQ

## Overview

This page covers application manifest configuration and permission handling in Connect IQ development.

## Key Topics

The manifest file defines core application properties including:
- App identification and metadata
- Required permissions
- Target devices
- API level requirements
- App type and capabilities

## Permissions System

Connect IQ apps must declare permissions for accessing:
- Sensors and location data
- Network communications
- Storage and file system
- User data and health metrics
- Device features

**Note:** For complete manifest and permissions documentation with examples, visit the Garmin Developers portal at developer.garmin.com/connect-iq/core-topics/manifest-and-permissions/
