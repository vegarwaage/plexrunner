# Sensors - Connect IQ

## Overview

This page covers sensor data access in Connect IQ applications.

## Key Topics

Sensor capabilities include:
- Built-in sensors (accelerometer, gyroscope, etc.)
- External sensor connectivity
- Sensor data streaming
- Data filtering and processing
- Sensor permissions
- Power management

## Common Sensor Types

- Heart Rate
- Accelerometer
- Gyroscope
- Barometer
- Compass
- Temperature
- Pulse Oximeter

## Related Topics

- ANT and ANT+
- Bluetooth Low Energy
- Positioning
- Activity Recording
