# Publishing to the Store - Connect IQ

## Overview

This page covers app submission processes for the Connect IQ Store.

## Key Topics

Publishing includes:
- App preparation
- Store listing creation
- Screenshot requirements
- Metadata configuration
- Review process
- Updates and versioning

## Submission Requirements

- Tested .iq file
- App description
- Screenshots
- Privacy policy (if applicable)
- Pricing information
- Device compatibility
