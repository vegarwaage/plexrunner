# Compatible Devices - Connect IQ

## Overview

This page documents Garmin devices compatible with Connect IQ development, organized by device type with specifications including screen resolution, shape, technology, and API level support.

## Device Categories

### Watches/Wearables

The primary category includes smartwatches and wearable devices with varying capabilities:

**Screen Specifications:**
- Resolution ranges: 163×156 to 454×454 pixels
- Shapes: round, semi-round, semi-octagon
- Technologies: AMOLED, Memory-In-Pixel (2, 8, 16, or 64 colors)

**API Level Support:**
Devices support Connect IQ versions from 1.2 through 5.2, with newer models generally supporting higher API levels.

### Key Device Lines

**fēnix® Series** (Multisport/Outdoor)
- Latest: fēnix® 8 series (43mm, 47mm, 51mm variants)
- Resolution: 416×416 to 454×454 AMOLED
- API Level: 5.2

**Forerunner® Series** (Running/Training)
- Latest: Forerunner® 965, 970
- Resolution: 454×454 AMOLED
- API Level: 5.2

**Instinct® Series** (Rugged/Outdoor)
- Latest: Instinct® 3 AMOLED, Solar variants
- Resolution: 176×176 to 416×416
- API Level: 5.1-5.2

**D2™ Series** (Aviation/Professional)
- Resolution: 218×218 to 454×454
- API Level: 1.4-5.2

### Outdoor Handhelds & Edge Devices

The listing includes dedicated outdoor navigation and cycling computers with varying display technologies.

## Filtering Options

The interface provides filtering by:
- Device type (Watches/Wearables, Outdoor Handhelds, Edge)
- Supported Connect IQ version (1.2 through 5.2)

## Device Selection Criteria

When developing Connect IQ applications, reference this table to verify:
- Target device availability and API level
- Screen resolution and shape constraints
- Display technology capabilities (particularly for color depth)
