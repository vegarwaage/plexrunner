# Connect IQ SDK - Get Started Guide

## Overview

The Connect IQ SDK enables developers to create applications for Garmin devices. The latest version is **8.3.0** (updated October 22, 2025).

## Installation Options

### SDK Manager (Recommended)

Three platform-specific installers are available:

- **Windows**: connectiq-sdk-manager-windows.zip
- **Mac**: connectiq-sdk-manager.dmg
- **Linux**: connectiq-sdk-manager-linux.zip

**Setup Steps:**
1. Download the appropriate installer
2. Launch the SDK manager
3. Complete first-time setup
4. Use manager to download latest SDK and device simulators
5. Activate new SDK version when prompted
6. Close manager

### Visual Studio Code Extension

Developers using VS Code can install the Monkey C extension:

1. Open Extensions Marketplace (View > Extension)
2. Search for "Monkey C"
3. Install Garmin's official extension
4. Restart VS Code
5. Run "Monkey C: Verify Installation" via command palette (Ctrl+Shift+P)

### Direct Download

A direct download option exists for developers not using VS Code, though it excludes the extension.

## Mobile SDK Access

**Android**: Available via Maven Central repository with sample app on GitHub

**iOS**: Available via GitHub with accompanying example application

## SDK Capabilities

The 8.3.0 version supports Connect IQ versions 1.x, 2.x, and 3.x across compatible Garmin devices.

## License Agreement

All downloads require accepting Garmin's developer agreement, which covers application development rights, distribution terms, monetization rules, and intellectual property ownership.
