# Connect IQ SDK Overview

## Latest Version
**Connect IQ 8.3.0** (Updated: October 22, 2025)

## Platform Purpose
Connect IQ enables developers to build wearable applications leveraging Garmin device sensors and features. The SDK exposes APIs and ANT+ profiles to reach millions of users across Garmin's device ecosystem.

## Available App Types

### Audio Content Provider Apps
- Sync audio content from third-party providers
- Manage downloaded audio content
- Customize audio playback experience

### Data Fields
- Customize built-in features
- Present activity information creatively
- Access ANT+ sensors and generic ANT channels
- Capture, display, record, and share data with FIT recording

### Device Apps
- Extended device access
- Read and record FIT files
- Access accelerometer data
- ANT+ sensor integration
- GPS and sensor data leverage
- BLE communication with mobile phones/Internet
- OAuth-authenticated web service connections

### Widgets
- At-a-glance information display
- Custom cards in main carousel
- User interaction and custom interfaces
- ANT+ sensor access
- GPS and sensor data
- BLE mobile communication
- OAuth web service integration

### Watch Faces
- Customize default watch face appearance
- Build colorful, always-on designs
- Drawing toolkit, custom fonts, and bitmaps

## Development Language
The platform uses **Monkey C**, a familiar programming language designed for Connect IQ development.
