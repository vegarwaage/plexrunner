# Toybox.UserProfile Module

## Overview

The UserProfile module provides APIs for accessing user profile data in Connect IQ applications.

## Key Features

- User settings
- Profile data
- Heart rate zones
- Personal records
- User preferences
- Gender and age

## Common Use Cases

- Personalized apps
- Training zones
- Performance tracking
- User customization
- Profile-based features

**Note:** For complete Toybox.UserProfile API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/UserProfile.html
