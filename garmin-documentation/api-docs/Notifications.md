# Toybox.Notifications Module

## Overview

The Notifications module provides APIs for user notifications in Connect IQ applications.

## Key Features

- Notification display
- Alert management
- User interaction
- Notification actions
- Custom notifications

## Common Use Cases

- User alerts
- Event notifications
- Message display
- Action prompts
- System notifications

**Note:** For complete Toybox.Notifications API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Notifications.html
