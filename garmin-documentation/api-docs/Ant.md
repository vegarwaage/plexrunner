# Toybox.Ant Module

## Overview

The Ant module provides APIs for ANT wireless communication in Connect IQ applications.

## Key Features

- ANT channel management
- Message transmission
- Channel configuration
- Device pairing
- Data reception

## Common Use Cases

- Custom ANT sensor integration
- Wireless device communication
- Sensor data collection
- ANT device control

**Note:** For complete Toybox.Ant API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Ant.html
