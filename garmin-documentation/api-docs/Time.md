# Toybox.Time Module

## Overview

The Time module provides time and date APIs in Connect IQ applications.

## Key Features

- Current time
- Date formatting
- Time zones
- Time calculations
- Moment objects
- Duration calculations

## Common Use Cases

- Clock display
- Time calculations
- Date formatting
- Timestamp handling
- Time zone conversion

**Note:** For complete Toybox.Time API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Time.html
