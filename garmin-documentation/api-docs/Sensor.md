# Toybox.Sensor Module

## Overview

The Sensor module provides APIs for accessing device sensors in Connect IQ applications.

## Key Features

- Heart rate
- Accelerometer
- Barometer
- Compass
- Temperature
- Gyroscope
- Pulse oximeter

## Common Use Cases

- Sensor data collection
- Health monitoring
- Activity tracking
- Environmental data
- Motion detection

**Note:** For complete Toybox.Sensor API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Sensor.html
