# Toybox.Lang Module

## Overview

The Lang module provides core language features and utilities in Connect IQ applications.

## Key Features

- Basic types
- Exception classes
- Object utilities
- Type conversions
- Language constructs
- Reflection

## Common Use Cases

- Type checking
- Exception handling
- Object manipulation
- Type conversion
- Language utilities
