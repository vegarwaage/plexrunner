# Toybox.Application Module

## Overview

The Application module provides core application framework APIs in Connect IQ applications.

## Key Features

- Application lifecycle
- App base classes
- Properties management
- Storage access
- App state management
- Event handling

## Common Use Cases

- App initialization
- State persistence
- Property storage
- Lifecycle management
- Event handling

**Note:** For complete Toybox.Application API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Application.html
