# Toybox.PersistedLocations Module

## Overview

The PersistedLocations module provides APIs for managing saved locations in Connect IQ applications.

## Key Features

- Location storage
- Location retrieval
- Location metadata
- Favorites management
- Location search

## Common Use Cases

- Waypoint management
- Favorite locations
- Navigation apps
- Location tracking
- POI storage

**Note:** For complete Toybox.PersistedLocations API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/PersistedLocations.html
