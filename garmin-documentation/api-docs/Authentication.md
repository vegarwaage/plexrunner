# Toybox.Authentication Module

## Overview

The Authentication module provides APIs for OAuth authentication with web services in Connect IQ applications.

## Key Features

- OAuth 2.0 support
- Token management
- Authorization flows
- Secure authentication
- Credential storage

## Common Use Cases

- Web service integration
- API authentication
- User account linking
- Secure data access
- Third-party service integration

**Note:** For complete Toybox.Authentication API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Authentication.html
