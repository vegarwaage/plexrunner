# Toybox.Position Module

## Overview

The Position module provides APIs for GPS and location data in Connect IQ applications.

## Key Features

- GPS location data
- Position accuracy
- Satellite information
- Speed and heading
- Altitude data
- Location updates

## Common Use Cases

- GPS tracking
- Navigation
- Location display
- Distance calculation
- Geolocation

**Note:** For complete Toybox.Position API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Position.html
