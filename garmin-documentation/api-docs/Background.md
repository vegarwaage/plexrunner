# Toybox.Background Module

## Overview

The Background module provides APIs for background process execution in Connect IQ applications.

## Key Features

- Background tasks
- Temporal events
- Exit callbacks
- Process management
- Resource limits

## Common Use Cases

- Periodic data updates
- Background synchronization
- Scheduled tasks
- Data collection
- Background processing

**Note:** For complete Toybox.Background API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Background.html
