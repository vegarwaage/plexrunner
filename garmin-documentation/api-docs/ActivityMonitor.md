# Toybox.ActivityMonitor Module

## Overview

The ActivityMonitor module provides APIs for accessing daily activity monitoring data in Connect IQ applications.

## Key Features

- Step count
- Calorie tracking
- Distance traveled
- Active minutes
- Floors climbed
- Sleep data

## Common Use Cases

- Wellness tracking
- Daily activity display
- Goal monitoring
- Health metrics

**Note:** For complete Toybox.ActivityMonitor API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/ActivityMonitor.html
