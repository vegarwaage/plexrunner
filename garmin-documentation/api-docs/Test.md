# Toybox.Test Module

## Overview

The Test module provides testing framework APIs for Connect IQ applications.

## Key Features

- Unit testing
- Test assertions
- Test suites
- Mock objects
- Test utilities

## Common Use Cases

- Automated testing
- Unit tests
- Integration tests
- Quality assurance
- Test-driven development
