# Toybox.StringUtil Module

## Overview

The StringUtil module provides string manipulation utilities in Connect IQ applications.

## Key Features

- String formatting
- Number conversion
- String utilities
- Format specifiers
- Character encoding

## Common Use Cases

- Text formatting
- Number display
- String manipulation
- Data formatting
- Locale-aware formatting
