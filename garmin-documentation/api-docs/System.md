# Toybox.System Module

## Overview

The System module provides system-level APIs and device information in Connect IQ applications.

## Key Features

- Device information
- System stats
- Battery level
- Memory usage
- System settings
- Device capabilities

## Common Use Cases

- Device detection
- Resource monitoring
- Capability checking
- System information
- Platform detection

**Note:** For complete Toybox.System API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/System.html
