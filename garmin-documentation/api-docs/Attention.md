# Toybox.Attention Module

## Overview

The Attention module provides APIs for getting user attention through vibrations, tones, and alerts in Connect IQ applications.

## Key Features

- Vibration patterns
- Alert tones
- Visual indicators
- Attention levels
- Custom patterns

## Common Use Cases

- Notifications
- Alerts
- Timer alarms
- User feedback
- Event notifications

**Note:** For complete Toybox.Attention API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Attention.html
