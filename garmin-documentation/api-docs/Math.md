# Toybox.Math Module

## Overview

The Math module provides mathematical functions and utilities in Connect IQ applications.

## Key Features

- Trigonometric functions
- Logarithms
- Rounding operations
- Constants (PI, E)
- Min/max operations
- Random numbers

## Common Use Cases

- Calculations
- Geometry
- Statistics
- Data processing
- Algorithm implementation

**Note:** For complete Toybox.Math API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Math.html
