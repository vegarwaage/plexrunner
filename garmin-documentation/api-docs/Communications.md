# Toybox.Communications Module

## Overview

The Communications module provides APIs for network communication in Connect IQ applications.

## Key Features

- HTTP requests
- JSON parsing
- Web service calls
- Request/response handling
- Network callbacks
- Error management

## Common Use Cases

- API integration
- Data synchronization
- Weather services
- Web requests
- Cloud integration

**Note:** For complete Toybox.Communications API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Communications.html
