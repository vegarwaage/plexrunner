# Toybox.Weather Module

## Overview

The Weather module provides APIs for accessing weather data in Connect IQ applications.

## Key Features

- Current conditions
- Forecast data
- Temperature
- Precipitation
- Wind information
- Weather alerts

## Common Use Cases

- Weather displays
- Watch faces with weather
- Activity planning
- Weather alerts
- Condition monitoring

**Note:** For complete Toybox.Weather API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Weather.html
