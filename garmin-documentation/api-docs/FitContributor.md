# Toybox.FitContributor Module

## Overview

The FitContributor module provides APIs for contributing custom data to FIT files in Connect IQ applications.

## Key Features

- Custom FIT fields
- Data recording
- Field definitions
- FIT file integration
- Data types

## Common Use Cases

- Custom activity data
- Sensor data recording
- Training metrics
- Performance data
- Custom analytics

**Note:** For complete Toybox.FitContributor API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/FitContributor.html
