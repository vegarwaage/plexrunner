# Toybox.Timer Module

## Overview

The Timer module provides timer functionality in Connect IQ applications.

## Key Features

- Periodic timers
- One-shot timers
- Timer callbacks
- Timer management
- Scheduled execution

## Common Use Cases

- UI updates
- Animation
- Periodic tasks
- Delayed execution
- Scheduled events

**Note:** For complete Toybox.Timer API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Timer.html
