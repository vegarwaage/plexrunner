# Toybox.AntPlus Module

## Overview

The AntPlus module provides APIs for ANT+ device profile support in Connect IQ applications.

## Key Features

- Heart rate monitors
- Bike speed/cadence sensors
- Power meters
- Fitness equipment
- Environment sensors
- Standard ANT+ profiles

## Common Use Cases

- ANT+ sensor integration
- Heart rate monitoring
- Power data collection
- Cadence tracking
- Multi-sensor support

**Note:** For complete Toybox.AntPlus API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/AntPlus.html
