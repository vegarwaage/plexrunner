# Toybox.Activity Module

## Overview

The Activity module provides APIs for accessing activity-related information in Connect IQ applications.

## Key Features

- Access current activity data
- Activity session information
- Sport and sub-sport types
- Activity metrics
- Activity state

## Common Use Cases

- Data field applications
- Activity tracking
- Workout monitoring
- Performance metrics

**Note:** For complete Toybox.Activity API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Activity.html
