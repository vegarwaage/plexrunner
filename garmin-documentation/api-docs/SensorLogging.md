# Toybox.SensorLogging Module

## Overview

The SensorLogging module provides APIs for high-frequency sensor data logging in Connect IQ applications.

## Key Features

- High-frequency logging
- Sensor data capture
- Accelerometer logging
- Data export
- Performance optimization

## Common Use Cases

- Research applications
- Detailed analysis
- Motion tracking
- Activity analysis
- Performance monitoring

**Note:** For complete Toybox.SensorLogging API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/SensorLogging.html
