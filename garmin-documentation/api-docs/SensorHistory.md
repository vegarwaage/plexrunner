# Toybox.SensorHistory Module

## Overview

The SensorHistory module provides APIs for accessing historical sensor data in Connect IQ applications.

## Key Features

- Historical data access
- Time-series data
- Data sampling
- Heart rate history
- Temperature history
- Pressure history

## Common Use Cases

- Data analysis
- Trend visualization
- Historical tracking
- Performance analysis
- Long-term monitoring

**Note:** For complete Toybox.SensorHistory API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/SensorHistory.html
