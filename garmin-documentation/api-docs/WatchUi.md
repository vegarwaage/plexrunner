# Toybox.WatchUi Module

## Overview

The WatchUi module provides user interface APIs for Connect IQ applications.

## Key Features

- View management
- Input delegates
- Menus and pickers
- Layouts
- Drawables
- Behaviors
- Page management

## Common Use Cases

- UI development
- View creation
- Input handling
- Menu systems
- Custom controls

**Note:** For complete Toybox.WatchUi API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/WatchUi.html
