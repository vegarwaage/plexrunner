# Toybox.Graphics Module

## Overview

The Graphics module provides APIs for rendering and visual elements in Connect IQ applications.

## Key Features

- Drawing context
- Shapes and paths
- Text rendering
- Image handling
- Color management
- Transformations
- Clipping

## Common Use Cases

- Custom UI rendering
- Watch face graphics
- Data visualization
- Animation
- Custom drawing

**Note:** For complete Toybox.Graphics API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Graphics.html
