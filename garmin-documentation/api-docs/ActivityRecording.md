# Toybox.ActivityRecording Module

## Overview

The ActivityRecording module provides APIs for recording fitness activities and generating FIT files in Connect IQ applications.

## Key Features

- Session recording
- FIT file generation
- Lap marking
- Custom data fields
- Activity summary
- Auto-pause functionality

## Common Use Cases

- Fitness app development
- Custom activity tracking
- Workout recording
- Data field recording

**Note:** For complete Toybox.ActivityRecording API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/ActivityRecording.html
