# Toybox.Media Module

## Overview

The Media module provides APIs for audio content and playback control in Connect IQ applications.

## Key Features

- Content providers
- Playback control
- Media library access
- Audio streaming
- Content metadata

## Common Use Cases

- Audio content apps
- Music control
- Podcast apps
- Media management
- Playback integration

**Note:** For complete Toybox.Media API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Media.html
