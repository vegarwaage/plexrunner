# Toybox.BluetoothLowEnergy Module

## Overview

The BluetoothLowEnergy module provides APIs for BLE communication in Connect IQ applications.

## Key Features

- BLE device scanning
- GATT services
- Characteristic operations
- Notifications/Indications
- Connection management
- UUID handling

## Common Use Cases

- BLE sensor integration
- Device communication
- Heart rate monitors
- Custom BLE devices
- Data synchronization

**Note:** For complete Toybox.BluetoothLowEnergy API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/BluetoothLowEnergy.html
