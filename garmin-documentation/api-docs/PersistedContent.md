# Toybox.PersistedContent Module

## Overview

The PersistedContent module provides APIs for managing persisted content in Connect IQ applications.

## Key Features

- Content storage
- Content iteration
- Content deletion
- Persistent data
- Content management

## Common Use Cases

- Data persistence
- Content caching
- Offline storage
- Data management
- Content lifecycle

**Note:** For complete Toybox.PersistedContent API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/PersistedContent.html
