# Toybox.Cryptography Module

## Overview

The Cryptography module provides APIs for cryptographic operations in Connect IQ applications.

## Key Features

- Hash functions
- Encryption
- Secure random
- HMAC
- Data signing

## Common Use Cases

- Data security
- Password hashing
- Secure communication
- Data integrity
- Authentication

**Note:** For complete Toybox.Cryptography API reference with classes, methods, and properties, visit the Garmin Developers portal at developer.garmin.com/connect-iq/api-docs/Toybox/Cryptography.html
