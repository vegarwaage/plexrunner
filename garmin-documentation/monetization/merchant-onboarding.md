# Merchant Onboarding - Monetization

## Overview

This page covers the merchant onboarding process for selling Connect IQ applications.

## Key Topics

Merchant onboarding includes:
- Account setup
- Tax information
- Payment details
- Verification process
- Legal agreements
- Banking information

- Valid developer account

- Compliance verification
